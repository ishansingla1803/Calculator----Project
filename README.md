# 🧮 Calculator (HTML, CSS, JavaScript)

A responsive and user-friendly advanced calculator built using HTML, CSS, and JavaScript. This project supports basic arithmetic operations along with scientific functions.

## 🚀 Features
- Basic operations (+, -, *, /, %)
- Scientific functions (sin, cos, log, sqrt)
- Keyboard input support
- Error handling for invalid expressions
- Clean and modern UI

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📌 How to Run
1. Download or clone the repository
2. Open 'calculator.html' in your browser
